Configuración — Google Sheets

Sigue estos pasos una sola vez para conectar el sistema con Google Sheets. Todos los reportes quedarán guardados permanentemente y accesibles desde cualquier dispositivo.

1
Crea el Google Sheet
Ve a sheets.new para crear una hoja nueva. Ponle el nombre "Mantenimiento CSL".
2
Abre el editor de Apps Script
En el Sheet: menú Extensiones → Apps Script. Se abre una nueva pestaña.
3
Pega el código del backend
Borra todo lo que hay en el editor. Pega el contenido del archivo Code.gs que descargaste. Guarda con Ctrl+S.
4
Ejecuta el setup inicial
En Apps Script: selecciona la función setupInicial en el menú desplegable y haz clic en ▶ Ejecutar. Acepta los permisos cuando te los pida.
5
Despliega como aplicación web
Clic en Implementar → Nueva implementación
Tipo: Aplicación web
Ejecutar como: Yo
Quién tiene acceso: Cualquier usuario
Clic en Implementar → copia la URL que aparece.
6
Pega la URL aquí y conecta
Pega la URL del paso anterior:

Una vez conectado, la URL se guarda en tu navegador. No necesitas repetir este proceso.
